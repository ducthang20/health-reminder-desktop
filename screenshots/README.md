# Screenshots

Thư mục này chứa screenshots của ứng dụng.

## Cần có các file sau:

1. `main-window.png` - Screenshot của cửa sổ chính
2. `notification.png` - Screenshot của popup thông báo
3. `dark-mode.png` - Screenshot giao diện dark mode (optional)
4. `emoji-picker.png` - Screenshot emoji picker (optional)
5. `color-picker.png` - Screenshot color picker (optional)

## Kích thước khuyến nghị:

- Width: 800-1200px
- Format: PNG
- Nên có border/shadow để dễ nhìn trên GitHub

## Cách chụp screenshot:

1. Chạy app: `npm start`
2. Sử dụng công cụ chụp màn hình của OS
3. Crop và chỉnh sửa nếu cần
4. Lưu vào thư mục này
